# Outdoor Digital da Igreja (Realidade Aumentada)

Uma aplicação simples de Realidade Aumentada baseada na web (WebAR) que permite aos usuários visualizar um modelo 3D de um outdoor em seu ambiente real.

## 🌟 Funcionalidades

- **Visualização de Modelo 3D**: Interação com um modelo `.glb` de alta qualidade de um outdoor de igreja.
- **Integração WebAR**: Utiliza o componente `<model-viewer>` do Google para projetar o modelo 3D no espaço do usuário via WebXR, Scene Viewer ou Quick Look.
- **Design Responsivo**: Funciona em navegadores de celular e desktop (AR requer dispositivos móveis compatíveis).

## 🚀 Tecnologias Utilizadas

- **HTML5/CSS3**: Estrutura principal e estilização premium com tema escuro.
- **[Model-Viewer](https://modelviewer.dev/)**: Um componente web para renderizar modelos 3D interativos.
- **GLB/glTF**: Formato de arquivo 3D padrão para a web.

## 📲 Como Usar

1.  Abra o arquivo `index.html` em um navegador web.
2.  Em um dispositivo móvel compatível, clique no botão **"✨ VER NO MEU ESPAÇO"**.
3.  Aponte sua câmera para uma superfície plana (chão) para posicionar o outdoor.
4.  Interaja com o modelo girando ou redimensionando-o na tela.

## 📂 Estrutura do Projeto

- `index.html`: O arquivo principal da aplicação contendo a lógica de AR e estilos.
- `billboard.glb`: O modelo 3D utilizado na aplicação.
